## 二、域名选择建议

### 域名命名原则

1. **简短易记**
   - ✅ 好：zilo.com
   - ❌ 差：zilo-influencer-management-system.com

2. **与品牌相关**
   - ✅ 好：zilo.com, zilohq.com
   - ❌ 差：abc123.com

3. **避免特殊字符**
   - ✅ 好：zilo.com
   - ❌ 差：zi-lo.com, zi_lo.com

4. **选择合适的后缀**
   - 推荐：`.com` > `.cn` > `.com.cn` > `.net`
   - 避免：`.xyz`, `.top`, `.club`（看起来不专业）

### 为 Zilo 推荐的域名

按优先级排序：

1. **zilo.com** ⭐⭐⭐⭐⭐
   - 最佳选择，简短、专业
   - 可能已被注册，需要查询

2. **zilohq.com** ⭐⭐⭐⭐⭐
   - HQ = Headquarters（总部）
   - 很多 SaaS 公司使用这种格式

3. **getzilo.com** ⭐⭐⭐⭐
   - "Get" 前缀很流行（如 getbootstrap.com）
   - 容易记忆

4. **zilo.cn** ⭐⭐⭐⭐
   - 中国域名，价格便宜
   - 适合国内市场

5. **usezilo.com** ⭐⭐⭐⭐
   - "Use" 前缀也很流行
   - 强调产品的使用

6. **ziloapp.com** ⭐⭐⭐
   - 明确表示是应用
   - 较长但清晰

## 三、腾讯云域名购买步骤

### 步骤 1：注册腾讯云账号

1. 访问 https://cloud.tencent.com/
2. 点击右上角"注册"
3. 使用手机号或微信注册
4. 完成手机验证

### 步骤 2：实名认证（必须）

1. 登录后，进入"账号中心"
2. 点击"实名认证"
3. 选择认证类型：
   - **个人认证**：需要身份证
   - **企业认证**：需要营业执照（推荐，更专业）

**企业认证材料**：
- 营业执照扫描件
- 法人身份证
- 对公账户信息（可选）

**认证时间**：1-3 个工作日

### 步骤 3：查询域名

1. 访问 https://dnspod.cloud.tencent.com/
2. 在搜索框输入想要的域名（如：zilo）
3. 点击"查询"
4. 查看哪些后缀可以注册

**示例**：
```
zilo.com     - 已被注册 ❌
zilohq.com   - 可以注册 ✅
getzilo.com  - 可以注册 ✅
zilo.cn      - 可以注册 ✅
```

### 步骤 4：购买域名

1. 选择可注册的域名
2. 点击"加入购物车"
3. 选择购买年限（建议：3-5年，有折扣）
4. 勾选"域名信息模板"（首次需创建）
5. 点击"立即购买"
6. 完成支付

**费用示例**：
- 1年：¥55
- 3年：¥165（平均 ¥55/年）
- 5年：¥275（平均 ¥55/年）

### 步骤 5：创建域名信息模板

**个人模板**：
- 姓名
- 身份证号
- 联系电话
- 电子邮箱
- 通讯地址

**企业模板**：
- 公司名称
- 营业执照号
- 法人姓名
- 联系电话
- 电子邮箱
- 公司地址

**重要**：信息必须真实，用于备案！

### 步骤 6：域名实名认证

购买后需要进行域名实名认证：

1. 进入"域名管理"
2. 找到刚购买的域名
3. 点击"实名认证"
4. 上传证件照片
5. 等待审核（1-3 个工作日）

**注意**：域名实名认证通过后才能备案！

## 四、域名备案流程

### 什么是备案？

在中国大陆，所有网站都必须进行 ICP 备案才能访问。

**备案时间**：7-20 个工作日

### 备案类型

1. **首次备案**：第一次备案
2. **新增网站**：已有备案，添加新域名
3. **接入备案**：域名已备案，更换服务商

### 备案材料（企业）

1. **营业执照**扫描件
2. **法人身份证**正反面
3. **网站负责人身份证**正反面
4. **网站负责人照片**（腾讯云提供电子幕布）
5. **域名证书**（腾讯云自动生成）
6. **授权书**（如果负责人不是法人）

### 备案材料（个人）

1. **身份证**正反面
2. **本人照片**（腾讯云提供电子幕布）
3. **域名证书**

### 腾讯云备案步骤

#### 第一步：进入备案系统

1. 登录腾讯云控制台
2. 搜索"网站备案"
3. 点击"开始备案"

#### 第二步：填写备案信息

1. **主体信息**
   - 企业：营业执照信息
   - 个人：身份证信息

2. **网站信息**
   - 网站名称：Zilo 达人营销执行系统
   - 网站域名：你购买的域名
   - 服务内容：选择"其他"
   - 网站描述：简要说明网站用途

3. **负责人信息**
   - 姓名、电话、邮箱
   - 上传身份证照片

#### 第三步：上传资料

1. 上传营业执照
2. 上传身份证
3. 拍摄负责人照片（使用腾讯云电子幕布）

**电子幕布使用**：
- 腾讯云会提供一个小程序
- 扫码后按提示拍照
- 无需实体幕布

#### 第四步：提交审核

1. 腾讯云初审（1-2 个工作日）
2. 管局审核（7-20 个工作日）
3. 备案成功，获得备案号

#### 第五步：备案成功

收到备案号后：
1. 在网站底部添加备案号
2. 链接到 https://beian.miit.gov.cn/

## 五、域名解析配置

备案通过后，需要配置域名解析：

### 在腾讯云 DNS 控制台

1. 进入"域名管理"
2. 点击域名后的"解析"
3. 添加记录：

```
类型: A
主机记录: @
记录值: 你的服务器公网IP
TTL: 600
```

```
类型: A
主机记录: www
记录值: 你的服务器公网IP
TTL: 600
```

### 等待生效

- 国内：10分钟 - 2小时
- 国外：24-48小时

## 七、常见问题

### Q1: 域名必须在腾讯云购买吗？

A: 不是必须，但强烈建议。原因：
- 同平台管理方便
- 备案流程更简单
- DNS 解析更快

### Q2: 可以先买域名，后备案吗？

A: 可以。流程：
1. 购买域名
2. 域名实名认证
3. 购买服务器
4. 提交备案
5. 备案通过后解析域名

### Q3: 备案期间网站能访问吗？

A: 不能。备案通过前：
- 可以用 IP 访问（仅限测试）
- 不能用域名访问
- 建议在本地或国外服务器测试

### Q4: 个人备案和企业备案有什么区别？

A: 
- **个人备案**：只能做个人网站，不能有商业内容
- **企业备案**：可以做商业网站，更专业

**建议**：如果有营业执照，选企业备案

### Q5: 域名可以转移吗？

A: 可以，但有限制：
- 注册满 60 天后才能转出
- 需要获取转移密码
- 转移期间域名会锁定

### Q6: 一个服务器可以备案多个域名吗？

A: 可以。腾讯云轻量应用服务器：
- 每台服务器可以备案 5 个域名
