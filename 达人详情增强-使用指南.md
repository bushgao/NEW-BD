## 📋 详情面板功能说明

### 标签页1: 基本信息

**查看内容**:
- ✅ 达人昵称
- ✅ 平台类型（抖音/快手/小红书等）
- ✅ 平台账号ID
- ✅ 手机号
- ✅ 微信号
- ✅ 粉丝数
- ✅ 类目标签
- ✅ 自定义标签
- ✅ 备注信息
- ✅ 创建时间
- ✅ 更新时间

**使用场景**:
- 快速了解达人的基本档案
- 查看联系方式准备沟通
- 确认达人的类目和定位

**操作提示**:
- 如需修改信息，点击"编辑"按钮
- 如需添加标签，点击标签图标

---

### 标签页2: ROI数据

**关键指标卡片**:

1. **平均ROI**
   - 显示该达人所有合作的平均投资回报率
   - 绿色表示正ROI，红色表示负ROI
   - 帮助评估达人的整体商业价值

2. **总GMV**
   - 显示该达人带来的总销售额
   - 蓝色显示，单位为人民币
   - 反映达人的带货能力

3. **总成本**
   - 显示与该达人合作的总投入
   - 包括样品成本、推广费用等
   - 用于计算ROI和利润

4. **成功率**
   - 显示ROI为正的合作占比
   - 绿色表示高成功率（>50%）
   - 橙色表示中等成功率（≤50%）

**最佳合作样品**:
- 显示ROI最高的合作样品信息
- 包括样品名称、ROI和GMV
- 帮助识别最佳合作机会

**合作统计**:
- 合作次数：总共合作的次数
- 成功合作：ROI为正的合作次数

**使用场景**:
- 评估达人的商业价值
- 决策是否继续合作
- 识别最佳合作样品类型
- 对比不同达人的表现

---

### 标签页3: 合作历史

**表格列说明**:

| 列名 | 说明 |
|------|------|
| 阶段 | 合作当前所处的阶段（建联/已联系/已报价等） |
| 样品 | 合作使用的样品名称 |
| 商务 | 负责该合作的商务人员 |
| GMV | 该合作产生的销售额 |
| ROI | 该合作的投资回报率 |
| 创建时间 | 合作记录的创建日期 |

**阶段标签颜色**:
- 🔵 建联 - 灰色
- 🔵 已联系 - 蓝色
- 🔵 已报价 - 青色
- 🟠 已寄样 - 橙色
- 🟣 已排期 - 紫色
- 🟢 已发布 - 绿色
- ✅ 已复盘 - 成功绿

**ROI颜色编码**:
- 🟢 绿色：ROI > 100%（盈利超过100%）
- 🟠 橙色：0% < ROI ≤ 100%（有盈利但不到100%）
- 🔴 红色：ROI ≤ 0%（亏损）

**使用场景**:
- 查看与达人的完整合作记录
- 了解历史合作的结果
- 追踪合作进展
- 分析合作趋势

**操作提示**:
- 表格支持滚动查看更多记录
- 按创建时间倒序排列（最新的在前）

## 💡 使用技巧

### 技巧1: 快速评估达人价值

1. 打开达人详情
2. 切换到"ROI数据"标签页
3. 查看以下指标：
   - 平均ROI > 50%：优质达人 ⭐⭐⭐
   - 平均ROI 0-50%：可合作达人 ⭐⭐
   - 平均ROI < 0%：需谨慎 ⭐

### 技巧2: 识别最佳合作样品

1. 打开达人详情
2. 切换到"ROI数据"标签页
3. 查看"最佳合作样品"卡片
4. 记录该样品信息
5. 在后续合作中优先推荐该类样品

### 技巧3: 分析合作趋势

1. 打开达人详情
2. 切换到"合作历史"标签页
3. 从上到下查看合作记录
4. 观察ROI变化趋势：
   - 上升趋势：合作越来越好 📈
   - 下降趋势：需要调整策略 📉
   - 波动较大：不稳定，需谨慎 📊

### 技巧4: 准备沟通前的信息收集

1. 打开达人详情
2. 查看"基本信息"获取联系方式
3. 查看"合作历史"了解过往合作
4. 查看"ROI数据"评估合作价值
5. 准备针对性的沟通话术

## 🎯 常见场景

### 场景1: 决定是否继续合作

**步骤**:
1. 打开达人详情
2. 查看ROI数据标签页
3. 评估指标：
   - 平均ROI是否为正？
   - 成功率是否超过50%？
   - 总GMV是否达到预期？
4. 查看合作历史标签页
5. 确认最近几次合作的表现
6. 做出决策

**决策参考**:
- ✅ 继续合作：平均ROI > 0，成功率 > 50%
- ⚠️ 谨慎合作：平均ROI接近0，成功率 30-50%
- ❌ 暂停合作：平均ROI < 0，成功率 < 30%

### 场景2: 选择合作样品

**步骤**:
1. 打开达人详情
2. 查看ROI数据标签页
3. 找到"最佳合作样品"
4. 记录样品名称和ROI
5. 在新合作中优先推荐该样品或同类样品

### 场景3: 准备商务汇报

**步骤**:
1. 打开达人详情
2. 收集以下数据：
   - 基本信息：昵称、平台、粉丝数
   - ROI数据：平均ROI、总GMV、成功率
   - 合作历史：合作次数、最近合作情况
3. 整理成汇报材料
4. 突出亮点和改进点

## ⚠️ 注意事项

### 数据准确性

1. **ROI计算说明**:
   - ROI = (GMV / 成本 - 1) × 100%
   - 只统计已复盘的合作记录
   - 未录入结果的合作不计入统计

2. **成功率计算说明**:
   - 成功定义：ROI > 0
   - 成功率 = 成功合作数 / 总合作数 × 100%

3. **数据更新**:
   - 数据实时从数据库获取
   - 每次打开详情面板都会刷新数据
   - 如有疑问，可关闭后重新打开

### 权限说明

- ✅ 所有商务人员都可以查看达人详情
- ✅ 基础商务只能查看自己创建的达人
- ✅ 高级商务和团队主管可以查看所有达人
- ❌ 编辑功能需要"达人管理"权限

## 🔧 故障排除

### 问题1: 详情面板打不开

**可能原因**:
- 网络连接问题
- 后端服务异常

**解决方法**:
1. 检查网络连接
2. 刷新页面重试
3. 联系技术支持

### 问题2: 数据显示为空

**可能原因**:
- 该达人确实没有合作记录
- 数据权限限制

**解决方法**:
1. 确认该达人是否有合作记录
2. 检查自己的数据查看权限
3. 联系管理员确认

### 问题3: ROI数据不准确

**可能原因**:
- 合作结果未录入
- 成本数据缺失

**解决方法**:
1. 确认所有合作都已录入结果
2. 检查成本数据是否完整
3. 联系负责商务补充数据
