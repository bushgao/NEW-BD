# ROI 分析图表 - 快速使用指南

**功能**: 工厂老板 Dashboard 的 ROI 分析图表  
**版本**: 1.0  
**更新时间**: 2026年1月6日

---

## 📖 功能介绍

ROI 分析图表帮助工厂老板快速了解：
- 各商务的 ROI 表现
- 成本构成情况
- 成本与收益的关系

支持三种可视化方式：
1. **柱状图** - 各商务 ROI 对比
2. **饼图** - 成本构成分析
3. **散点图** - 成本-收益关系

---

## 🚀 快速开始

### 1. 访问 Dashboard

登录工厂老板账号后，进入 Dashboard 页面。

### 2. 找到 ROI 分析图表

在页面中找到 "ROI 分析" 卡片，位于趋势图表下方。

### 3. 切换图表类型

点击右上角的切换按钮，选择想要查看的图表类型：
- **柱状图** - 对比各商务的业绩
- **饼图** - 查看成本构成
- **散点图** - 分析成本-收益关系

---

## 📊 图表说明

### 柱状图 - 各商务 ROI 对比

**用途**: 快速对比各商务的业绩表现

**展示内容**:
- 蓝色柱：总 GMV
- 红色柱：总成本
- 绿色柱：ROI

**如何阅读**:
- 柱子越高，数值越大
- GMV 和成本使用左侧 Y 轴
- ROI 使用右侧 Y 轴
- 鼠标悬停查看详细数值

**示例**:
```
张三: GMV ¥10,000, 成本 ¥5,000, ROI 2.0
李四: GMV ¥8,000, 成本 ¥4,000, ROI 2.0
王五: GMV ¥6,000, 成本 ¥4,000, ROI 1.5
```

**分析建议**:
- ROI > 1: 盈利
- ROI = 1: 盈亏平衡
- ROI < 1: 亏损
- 关注 ROI 低的商务，分析原因

### 饼图 - 成本构成分析

**用途**: 了解成本的主要构成

**展示内容**:
- 样品成本（蓝色）
- 合作成本（绿色）
- 其他成本（橙色）

**如何阅读**:
- 每个扇区显示百分比
- 中心显示总成本
- 鼠标悬停查看详细金额

**示例**:
```
样品成本: ¥2,000 (20%)
合作成本: ¥8,000 (80%)
总成本: ¥10,000
```

**分析建议**:
- 样品成本占比过高：考虑优化寄样策略
- 合作成本占比过高：关注合作费用控制
- 对比不同时期的成本构成变化

### 散点图 - 成本-收益关系

**用途**: 分析成本投入与收益产出的关系

**展示内容**:
- X 轴：成本
- Y 轴：收益（GMV）
- 气泡大小：ROI 值
- 气泡颜色：
  - 绿色：盈利（ROI ≥ 1）
  - 红色：亏损（ROI < 1）
- 虚线：盈亏平衡线（成本 = 收益）

**如何阅读**:
- 点在虚线上方：盈利
- 点在虚线上：盈亏平衡
- 点在虚线下方：亏损
- 气泡越大，ROI 越高
- 鼠标悬停查看商务名称和详细数据

**示例**:
```
张三: 成本 ¥5,000, 收益 ¥10,000, ROI 2.0 (绿色大气泡)
李四: 成本 ¥4,000, 收益 ¥8,000, ROI 2.0 (绿色大气泡)
王五: 成本 ¥4,000, 收益 ¥3,000, ROI 0.75 (红色小气泡)
```

**分析建议**:
- 关注红色气泡（亏损商务）
- 分析高成本低收益的原因
- 学习高 ROI 商务的经验
- 优化成本投入策略

## 💡 使用技巧

### 1. 综合分析

结合三种图表进行综合分析：
1. **柱状图** - 找出 ROI 最高和最低的商务
2. **饼图** - 了解成本主要花在哪里
3. **散点图** - 分析成本投入是否合理

### 2. 定期查看

建议每周查看一次 ROI 分析：
- 周一：查看上周数据
- 发现问题及时调整
- 跟踪改进效果

### 3. 对比分析

- 对比不同商务的 ROI
- 找出最佳实践
- 推广成功经验

### 4. 问题诊断

**ROI 低的可能原因**:
- 寄样成本过高
- 合作费用过高
- 转化率低
- 选品不当

**改进建议**:
- 优化寄样策略
- 谈判更好的合作价格
- 提高跟进质量
- 改进选品方法

## 📈 数据说明

### ROI 计算公式
```
ROI = 总GMV / 总成本
```

### 数据来源
- **总 GMV**: 所有已发布合作的销售额总和
- **总成本**: 寄样成本 + 合作费用
- **合作数量**: 已完成的合作数量

### 数据更新
- 实时更新
- 刷新页面获取最新数据

## ❓ 常见问题

### Q1: 为什么没有数据？
**A**: 可能的原因：
- 还没有完成的合作
- 还没有录入合作结果
- 商务人员还没有创建合作

**解决方法**:
- 等待合作完成
- 及时录入合作结果
- 鼓励商务人员积极开展合作

### Q2: 为什么某个商务没有显示？
**A**: 可能的原因：
- 该商务还没有完成的合作
- 该商务的合作还没有录入结果

**解决方法**:
- 等待该商务完成合作
- 提醒该商务录入结果

### Q3: ROI 数据不准确？
**A**: 请检查：
- 合作结果是否正确录入
- GMV 和成本数据是否准确
- 是否有遗漏的合作

**解决方法**:
- 核对合作结果数据
- 更正错误数据
- 补充遗漏的合作

### Q4: 如何提高 ROI？
**A**: 建议：
- 优化寄样策略，降低寄样成本
- 谈判更好的合作价格
- 提高合作转化率
- 选择更合适的达人
- 改进跟进方法
