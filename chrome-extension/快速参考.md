# Chrome 插件快速参考

## 🚀 3 步开始使用

### 1️⃣ 安装插件

```
chrome://extensions/ 
→ 开发者模式 
→ 加载已解压的扩展程序 
→ 选择 chrome-extension 文件夹
```

### 2️⃣ 配置插件

```
Zilo 系统登录 
→ F12 > Application > Local Storage > 复制 token 
→ 点击插件图标 > 配置设置 
→ 填写 API 地址和 Token 
→ 保存
```

### 3️⃣ 开始采集

```
打开抖音精选联盟达人详情页 
→ 点击"添加到 Zilo"按钮 
→ 等待采集完成 
→ 在 Zilo 系统查看结果
```

## 🐛 常见问题

### 按钮没出现？

- 确保在达人详情页（URL 包含 `daren-profile`）
- 刷新页面
- 检查插件是否启用

### 采集失败？

- 检查后端服务是否运行
- 更新 Token
- 查看控制台错误

### 提示未配置？

- 重新配置 Token
- 确保 Token 完整复制
