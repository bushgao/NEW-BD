# 插件图标

请在此文件夹中放置以下尺寸的图标文件：

- `icon16.png` - 16x16 像素（工具栏图标）
- `icon48.png` - 48x48 像素（扩展管理页面）
- `icon128.png` - 128x128 像素（Chrome 网上应用店）

## 设计建议

- 使用 Zilo 品牌色：紫色渐变 (#667eea 到 #764ba2)
- 简洁的 "Z" 字母标识
- 背景透明或白色
- 圆角矩形或圆形背景

## 临时方案

如果暂时没有图标，可以使用在线工具快速生成：
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

或者使用纯色占位图标：
1. 创建一个紫色背景的正方形
2. 在中间放置白色的 "Z" 字母
3. 导出为 PNG 格式，分别保存为 16x16、48x48、128x128 三个尺寸
