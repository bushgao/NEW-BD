# 自动点击眼睛图标功能

## 需求
用户点击"添加到 Zilo"按钮时，自动点击小眼睛图标显示联系方式，然后再采集。

## 实现方案

### 1. 查找眼睛图标元素
需要找到页面上的眼睛图标按钮，可能的选择器：
- 包含"eye"关键字的 class
- 包含"查看"、"显示"等文字的按钮
- SVG 图标

### 2. 自动点击流程
1. 点击"添加到 Zilo"按钮
2. 查找眼睛图标
3. 如果找到，自动点击
4. 等待 1-2 秒让联系方式显示
5. 提取联系方式
6. 发送到后端

### 3. 代码实现
在 `content.js` 的 `handleCollect()` 函数中：
1. 先尝试自动点击眼睛图标
2. 等待一段时间
3. 再提取达人信息

## 注意事项
- 需要处理眼睛图标找不到的情况
- 需要合理的等待时间（不能太短也不能太长）
- 需要在控制台输出调试信息
