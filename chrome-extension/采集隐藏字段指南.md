# 采集抖音达人隐藏手机号和微信号指南

## 方法1: 使用测试脚本（推荐）

### 步骤：

1. **打开抖音达人主页**
   - 在浏览器中打开任意抖音达人的主页
   - 例如：https://www.douyin.com/user/xxxxx

2. **打开浏览器控制台**
   - 按 `F12` 或右键点击页面 → "检查"
   - 切换到 "Console"（控制台）标签

3. **复制并运行测试脚本**
   - 打开文件：`chrome-extension/测试隐藏字段采集.js`
   - 复制全部内容
   - 粘贴到控制台中
   - 按 `Enter` 运行

4. **查看测试结果**
   - 脚本会自动查找：
     - ✅ 包含"手机"、"微信"关键词的元素
     - ✅ 眼睛图标（显示/隐藏按钮）
     - ✅ 隐藏的输入框
     - ✅ 被星号遮挡的文本
   
5. **根据结果调整采集策略**
   - 如果找到眼睛图标，脚本会提示如何点击
   - 如果找到隐藏字段，会显示其位置和内容

## 方法2: 手动查找（用于理解页面结构）

### 步骤：

1. **在达人主页找到联系方式区域**
   - 通常在左侧个人信息卡片中
   - 查找"手机"、"微信"等标签

2. **找到眼睛图标**
   - 在手机号或微信号旁边
   - 通常是一个 SVG 图标

3. **点击眼睛图标**
   - 手动点击查看隐藏内容
   - 观察页面变化

4. **在控制台运行以下命令查看变化**
   ```javascript
   // 查找所有包含手机号的元素
   document.querySelectorAll('*').forEach(el => {
     if (el.textContent.includes('手机') && el.textContent.length < 100) {
       console.log('手机号区域:', el.textContent, el);
     }
   });
   
   // 查找所有包含微信的元素
   document.querySelectorAll('*').forEach(el => {
     if (el.textContent.includes('微信') && el.textContent.length < 100) {
       console.log('微信号区域:', el.textContent, el);
     }
   });
   ```

## 方法3: 快速定位（一行命令）

在控制台运行以下命令，快速查看所有联系方式：

```javascript
// 快速查看所有可能的联系方式
Array.from(document.querySelectorAll('*'))
  .filter(el => {
    const text = el.textContent;
    return (text.includes('手机') || text.includes('微信')) && text.length < 200;
  })
  .forEach(el => {
    console.log('---');
    console.log('文本:', el.textContent.trim());
    console.log('元素:', el);
    console.log('HTML:', el.innerHTML.substring(0, 200));
  });
```
