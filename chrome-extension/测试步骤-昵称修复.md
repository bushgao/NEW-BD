## 🔄 第一步：重新加载插件

1. 打开 Chrome 扩展管理页面：`chrome://extensions/`
2. 找到"Zilo 达人采集助手"
3. **点击刷新图标 🔄**（在插件卡片上）
4. 确认看到"已更新"提示

## 🧪 第二步：测试采集功能

1. **刷新抖音达人详情页**
   - 在当前的达人详情页按 `F5` 刷新
   - 或者重新打开达人详情页

2. **打开浏览器控制台**
   - 按键盘 `F12`
   - 切换到 `Console` 标签

3. **点击"添加到 Zilo"按钮**
   - 观察控制台输出
   - 应该看到类似这样的日志：
     ```
     [Zilo] 找到昵称: XXX 选择器: span.auxo-dorami-atom-text
     [Zilo] 提取的达人信息: {nickname: "XXX", ...}
     ```

4. **查看结果**
   - ✅ 成功：按钮变成"✓ 已添加"，显示绿色
   - ❌ 失败：显示错误提示

## 📋 第三步：检查提取的信息

### 在控制台查看

找到这行日志：
```
[Zilo] 提取的达人信息: {...}
```

点击展开，检查各个字段：

| 字段 | 说明 | 期望值 |
|------|------|--------|
| `nickname` | 达人昵称 | ✅ 应该有值（不为空） |

## 🔍 第四步：如果其他字段为空

### 找到正确的选择器

1. **在达人详情页，右键点击要提取的信息**
   - 例如：抖音号、粉丝数、类目等
   
2. **选择"检查"或"检查元素"**
   - 浏览器会打开开发者工具并定位到该元素

3. **查看元素的 HTML 结构**
   - 找到包含文本的标签
   - 记录标签的 `class` 属性

4. **截图或复制 HTML 代码**
   - 将信息发给我，我会更新选择器

### 示例

如果粉丝数显示为"12.5万"，右键点击它：
```html
<span class="some-class-name">12.5万</span>
```

记录 `class="some-class-name"`，告诉我这个类名。
