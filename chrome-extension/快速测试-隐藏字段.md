# 快速测试：采集隐藏的手机号和微信号

## 🎯 目标
采集抖音达人主页中被隐藏的手机号和微信号（需要点击眼睛图标才能看到的信息）

---

## 📋 操作步骤

### 方法1: 自动采集（推荐）⭐

1. **打开抖音达人主页**
   ```
   在浏览器中打开任意抖音达人的主页
   确保页面完全加载
   ```

2. **打开控制台**
   ```
   按 F12 键
   或右键点击页面 → 选择"检查"
   切换到 "Console"（控制台）标签
   ```

3. **运行自动采集脚本**
   ```
   1. 打开文件: chrome-extension/自动采集隐藏字段.js
   2. 复制全部内容（Ctrl+A, Ctrl+C）
   3. 粘贴到控制台（Ctrl+V）
   4. 按 Enter 运行
   ```

4. **查看结果**
   ```
   脚本会自动：
   ✅ 查找联系方式区域
   ✅ 查找眼睛图标
   ✅ 自动点击眼睛图标
   ✅ 提取手机号和微信号
   ✅ 显示最终结果
   ```

---

### 方法2: 手动测试（用于调试）

如果自动脚本没有找到信息，可以手动测试：

1. **在控制台运行以下命令，查找联系方式区域：**
   ```javascript
   // 查找所有包含"手机"或"微信"的区域
   document.querySelectorAll('div').forEach(div => {
     const text = div.textContent;
     if ((text.includes('手机') || text.includes('微信')) && text.length < 500) {
       console.log('联系方式区域:', div);
       console.log('内容:', text);
       console.log('---');
     }
   });
   ```

2. **查找眼睛图标：**
   ```javascript
   // 查找所有 SVG 图标
   document.querySelectorAll('svg').forEach((svg, index) => {
     const parent = svg.closest('div');
     if (parent && (parent.textContent.includes('*') || 
                    parent.textContent.includes('手机') || 
                    parent.textContent.includes('微信'))) {
       console.log(`图标 ${index}:`, svg);
       console.log('父元素:', parent);
       console.log('文本:', parent.textContent);
     }
   });
   ```

3. **手动点击图标：**
   ```javascript
   // 如果找到了图标，手动点击（替换 INDEX 为实际的索引号）
   document.querySelectorAll('svg')[INDEX].click();
   ```

4. **点击后重新查看内容：**
   ```javascript
   // 查看点击后的变化
   document.querySelectorAll('div').forEach(div => {
     const text = div.textContent;
     if (text.includes('手机') || text.includes('微信')) {
       console.log('更新后的内容:', text);
     }
   });
   ```

## 🔍 如果没有找到信息

如果脚本没有找到手机号或微信号，可能的原因：

1. **页面结构不同**
   - 不同的达人主页可能有不同的布局
   - 需要查看实际的 HTML 结构

2. **信息不存在**
   - 该达人可能没有公开手机号或微信号

3. **需要登录**
   - 某些信息可能需要登录后才能查看

4. **动态加载**
   - 信息可能是通过 AJAX 动态加载的
