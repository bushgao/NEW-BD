## 🔍 常见问题

### Q1: 找不到眼睛图标
**A**: 可能联系方式已经显示，不需要点击。或者页面结构不同。

### Q2: 找到了但还是星号
**A**: 可能需要更长的等待时间，或者有多个眼睛图标没有全部点击。

### Q3: 手机号正常但微信号不行
**A**: 说明自动点击逻辑部分工作。可能微信号的眼睛图标位置特殊，需要调整查找逻辑。

### Q4: 手动点击可以，自动点击不行
**A**: 说明提取逻辑没问题，是自动点击的问题。需要优化 `clickEyeIcon()` 函数。
