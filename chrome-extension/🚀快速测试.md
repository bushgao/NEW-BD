# 🚀 快速测试微信号采集

## 📝 3步测试

### 1️⃣ 打开控制台
- 在达人详情页按 `F12`
- 切换到 `Console` 标签

### 2️⃣ 运行测试脚本
- 打开文件：`chrome-extension/一键测试微信号.js`
- 全选复制（Ctrl+A, Ctrl+C）
- 粘贴到控制台（Ctrl+V）
- 按回车

### 3️⃣ 查看结果
看到这个就是成功 ✅：
```
手机号: 15784011208(+6465) ✅
微信号: wxid_xxxxx ✅
🎉 完美！两个联系方式都采集成功！
```
