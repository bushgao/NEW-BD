# 从页面直接提取联系方式

## 问题分析
- URL 中没有 `ewid` 参数，无法调用 API
- 但是页面上已经显示了联系方式（点击眼睛图标后）
- 解决方案：直接从页面 DOM 中提取联系方式

## 提取策略

### 1. 查找包含"达人手机号"的文本
```javascript
// 查找所有文本节点
const allText = document.body.innerText;
const phoneMatch = allText.match(/达人手机号[：:]\s*([^\n]+)/);
if (phoneMatch) {
  phone = phoneMatch[1].trim();
}
```

### 2. 查找包含"达人微信号"的文本
```javascript
const wechatMatch = allText.match(/达人微信号[：:]\s*([^\n]+)/);
if (wechatMatch) {
  wechat = wechatMatch[1].trim();
}
```

### 3. 使用选择器查找
```javascript
// 查找包含特定文本的元素
const elements = document.querySelectorAll('*');
for (const el of elements) {
  const text = el.textContent;
  if (text.includes('达人手机号')) {
    // 提取手机号
  }
  if (text.includes('达人微信号')) {
    // 提取微信号
  }
}
```
