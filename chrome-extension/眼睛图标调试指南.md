# 眼睛图标调试指南

## 问题描述
联系方式（手机号和微信号）在达人详情页是隐藏的（显示为星号 `***********`），需要点击"眼睛图标"才能显示真实信息。目前自动点击功能不工作，需要找到正确的选择器。

## 调试步骤

### 第1步：运行一键测试脚本（推荐）

这是最简单的方法，脚本会自动尝试所有可能的策略并告诉你哪个有效。

1. 打开达人详情页（确保联系方式显示为星号）
2. 按 `F12` 打开浏览器控制台
3. 复制 `一键测试眼睛图标.js` 的全部内容
4. 粘贴到控制台并按回车运行
5. 查看输出结果

**如果成功：**
- 脚本会显示 "🎉 成功！联系方式已显示"
- 会告诉你成功的策略和元素信息
- 会提供建议的选择器
- **请把这个选择器告诉我，我会更新代码**

**如果失败：**
- 继续第2步

### 第2步：运行详细查找脚本

如果一键测试没有成功，使用这个脚本获取更详细的信息。

1. 刷新达人详情页（确保联系方式重新隐藏）
2. 打开控制台（F12）
3. 复制 `查找眼睛图标.js` 的全部内容
4. 粘贴到控制台并按回车运行
5. 脚本会列出所有可能的眼睛图标元素

**查看输出：**
- 脚本会找到多个可能的元素，每个都有一个测试命令
- 例如：`eyeIcon0.click()`, `svg1.click()`, `eye0.click()` 等

**逐个测试：**
1. 在控制台输入 `eyeIcon0.click()` 并回车
2. 观察页面上的联系方式是否从星号变成真实号码
3. 如果没变化，尝试下一个：`eyeIcon1.click()`
4. 继续测试直到找到正确的那个

**找到后：**
- 记录该元素的类名、ID 或其他特征
- 告诉我这些信息，我会更新代码

### 第3步：手动查找（如果脚本都失败）

1. 在达人详情页，找到显示联系方式的区域
2. 右键点击眼睛图标 → "检查元素"
3. 在开发者工具中查看该元素的：
   - 标签名（如 `svg`, `button`, `span`）
   - 类名（class）
   - ID
   - 父元素的类名
4. 截图或复制 HTML 代码告诉我

## 常见情况

### 情况1：联系方式本来就是显示的
如果页面上直接显示手机号和微信号（不是星号），说明：
- 不需要点击眼睛图标
- 当前的提取逻辑应该已经可以工作
- 可以直接测试采集功能

### 情况2：没有联系方式
如果页面上根本没有联系方式字段，说明：
- 该达人可能没有填写联系方式
- 或者需要特殊权限才能查看
- 尝试换一个有联系方式的达人测试

### 情况3：联系方式格式不同
如果联系方式不是显示为 "达人手机号：xxx" 的格式：
- 告诉我实际的文本格式
- 我会更新提取逻辑
