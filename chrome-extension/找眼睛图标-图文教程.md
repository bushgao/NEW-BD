# 找眼睛图标 - 超简单图文教程

## 第1步：打开达人详情页

1. 在抖音精选联盟，打开任意一个达人的详情页
2. 确保页面上的联系方式显示为星号 `***********`

## 第2步：打开浏览器控制台

按键盘上的 **F12** 键，或者：
- 右键点击页面 → 选择"检查"
- 或者按 `Ctrl + Shift + I` (Windows)
- 或者按 `Cmd + Option + I` (Mac)

## 第3步：运行查找脚本

1. 打开文件 `chrome-extension/超简单找眼睛.js`
2. 全选复制里面的所有代码（Ctrl+A, Ctrl+C）
3. 粘贴到控制台（Ctrl+V）
4. 按回车运行

## 第4步：测试图标

脚本会列出所有可能的图标，例如：

```
[0] SVG 图标:
   大小: 16x16px
   位置: x=500, y=300
   类名: icon-eye
   👉 测试命令: icon0.click()

[1] SVG 图标:
   大小: 20x20px
   位置: x=520, y=310
   类名: icon-visible
   👉 测试命令: icon1.click()
```

**逐个测试：**

1. 在控制台输入：`icon0.click()` 然后回车
2. 看页面上的联系方式：
   - ✅ 如果从 `***********` 变成了真实号码 → **成功！**
   - ❌ 如果还是星号 → 继续下一个

3. 试试：`icon1.click()` 然后回车
4. 继续试：`icon2.click()`, `icon3.click()` ...

## 第5步：告诉我结果

找到能显示联系方式的图标后，告诉我：

**"是 icon3"** 或 **"是 icon5"** 等等

我会根据这个信息更新代码。

## 如果所有 icon 都试过了还是不行

那就用手动方法：

1. 在页面上找到眼睛图标（就是你平时点击的那个）
2. **右键点击眼睛图标** → 选择"检查元素"
3. 浏览器会高亮显示这个元素的 HTML 代码
4. 截图发给我，或者告诉我：
   - 标签名（如 `<svg>`, `<button>`, `<span>`）
   - class 类名（如 `class="icon-eye"` 或 `class="auxo-icon"`）
   - 父元素的 class

## 常见问题

**Q: 控制台在哪里？**
A: 按 F12 键，会在浏览器底部或右侧出现一个面板

**Q: 怎么粘贴代码？**
A: 在控制台的输入框（通常有 `>` 符号），按 Ctrl+V 粘贴

**Q: 点击后没反应？**
A: 可能这个不是眼睛图标，继续试下一个 icon

**Q: 所有 icon 都试过了？**
A: 用手动方法，右键点击眼睛图标 → 检查元素
