# Zilo 达人采集助手 - Chrome 浏览器插件

一键采集抖音精选联盟达人信息到 Zilo 系统的 Chrome 浏览器插件。

## 功能特性

- ✨ 在抖音精选联盟达人详情页自动注入"添加到 Zilo"按钮
- 🚀 一键采集达人信息（昵称、抖音号、粉丝数、类目、等级等）
- 📊 实时统计采集成功和失败数量
- 🔐 安全的 Token 认证机制
- 💾 本地存储配置信息

## 安装步骤

### 1. 加载插件到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择 `chrome-extension` 文件夹
6. 插件安装成功！

### 2. 配置插件

1. 点击浏览器工具栏中的 Zilo 插件图标
2. 点击"⚙️ 配置设置"按钮
3. 填写以下信息：
   - **API 地址**：Zilo 后端 API 地址（默认：`http://localhost:3000/api`）
   - **登录令牌**：从 Zilo 系统获取的登录令牌

#### 如何获取登录令牌？

1. 在浏览器中打开 Zilo 系统（http://localhost:5173）
2. 使用您的账号登录
3. 按 `F12` 打开浏览器开发者工具
4. 切换到 `Application` 标签页
5. 在左侧菜单中找到 `Local Storage` > `http://localhost:5173`
6. 在右侧列表中找到 `token` 字段
7. 复制 `token` 的值（一长串字符串）
8. 粘贴到插件的"登录令牌"输入框中

4. 点击"💾 保存配置"按钮

## 使用方法

### 采集单个达人

1. 在 Chrome 浏览器中打开抖音精选联盟（https://buyin.jinritemai.com）
2. 登录您的账号
3. 进入达人详情页（例如：搜索达人 > 点击达人卡片）
4. 页面右上角会自动出现"添加到 Zilo"按钮
5. 点击按钮，插件会自动提取达人信息并保存到 Zilo 系统
6. 看到"✓ 已添加"提示表示采集成功

### 查看统计

1. 点击浏览器工具栏中的 Zilo 插件图标
2. 在弹出窗口中可以看到：
   - 连接状态
   - 已采集数量
   - 失败数量

## 采集的信息

插件会自动提取以下达人信息：

- **昵称**：达人的昵称
- **抖音号**：达人的抖音账号 ID
- **粉丝数**：达人的粉丝数量
- **类目**：达人的主推类目
- **等级**：达人的等级（如 LV2、LV3 等）
- **备注**：自动添加粉丝数和采集来源信息

## 常见问题

### 1. 按钮没有出现？

- 确保您在达人详情页（URL 包含 `daren-profile`）
- 刷新页面重试
- 检查插件是否已启用（`chrome://extensions/`）

### 2. 采集失败？

- 检查网络连接
- 确认 Zilo 后端服务正在运行（http://localhost:3000）
- 检查登录令牌是否正确
- 查看浏览器控制台（F12）的错误信息

### 3. 提示"未配置"？

- 点击插件图标 > 配置设置
- 填写 API 地址和登录令牌
- 保存配置

### 4. 提示"已达到达人数量上限"？

- 您的套餐已达到达人数量限制
- 请联系管理员升级套餐

```

├── manifest.json       # 插件配置文件
├── background.js       # 后台服务（处理 API 请求）
├── content.js          # 内容脚本（注入按钮和提取信息）
├── content.css         # 按钮样式
├── popup.html          # 弹出窗口 UI
├── popup.js            # 弹出窗口逻辑
├── icons/              # 插件图标
└── README.md           # 说明文档
```

## 安全说明

- 插件仅在抖音精选联盟域名下运行
- 登录令牌使用 Chrome 同步存储，加密保存
- 不会收集或上传任何个人隐私信息
- 所有数据仅保存到您自己的 Zilo 系统
