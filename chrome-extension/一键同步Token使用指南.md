# 🎯 一键同步 Token 使用指南

## 功能说明

现在切换账号超级简单！只需要点击两个按钮就能完成Token同步。

## 使用步骤（2步，5秒完成）

### 步骤1：在前端页面复制 Token（2秒）

1. 登录你想使用的账号（例如：测试002）
2. 点击右上角的 **"同步插件"** 按钮
3. 看到提示"✅ Token 已复制到剪贴板！"

### 步骤2：在插件中粘贴 Token（3秒）

1. 点击 Chrome 工具栏的 **Zilo 插件图标**
2. 点击 **"⚙️ 配置设置"** 按钮
3. 点击 **"📋 从剪贴板粘贴 Token"** 按钮
4. 点击 **"💾 保存配置"** 按钮
5. 看到提示"配置已保存"

## 完成！

现在采集的数据会进入当前登录的账号了！

## 常见问题

### Q1: 点击"同步插件"没反应？

**A**: 检查是否已登录。如果已登录，查看浏览器控制台是否有错误信息。

### Q2: 点击"从剪贴板粘贴"提示"剪贴板为空"？

**A**: 请先在前端页面点击"同步插件"按钮复制 Token。

### Q3: 提示"剪贴板内容不是有效的 Token"？

**A**: 
1. 确保先点击了前端的"同步插件"按钮
2. 确保没有复制其他内容覆盖了 Token
3. 重新点击"同步插件"按钮

### Q4: 保存后采集的数据还是进入错误的账号？

**A**: 
1. 确认前端页面显示的是正确的账号名
2. 重新执行一次同步流程
3. 重新加载插件（chrome://extensions/ → 刷新按钮）

## 技术说明

### 为什么需要两步操作？

由于浏览器安全限制，前端页面无法直接访问 Chrome 插件的存储。所以需要：
1. 前端页面将 Token 复制到剪贴板
2. 插件从剪贴板读取 Token

这是最简单、最安全的方案。

### Token 是什么？

Token 是登录凭证，类似于"钥匙"。每个账号有自己的 Token，插件使用这个 Token 来确定数据应该添加到哪个账号。
