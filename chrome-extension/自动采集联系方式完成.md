# 自动采集联系方式功能完成

## 功能说明

现在点击"添加到 Zilo"按钮时，插件会：
1. **自动查找并点击眼睛图标**
2. **等待 2 秒让联系方式显示**
3. **自动提取手机号和微信号**
4. **发送到后端保存**

## 实现细节

### 1. 自动点击眼睛图标
- 使用多个选择器查找眼睛图标
- 检查元素是否可见且可点击
- 自动点击后等待 2 秒

### 2. 查找策略
尝试以下选择器：
- `[class*="eye"]` - 包含 eye 的 class
- `[class*="Eye"]` - 包含 Eye 的 class
- `svg[class*="eye"]` - SVG 眼睛图标
- `button[class*="eye"]` - 按钮形式的眼睛图标
- `[aria-label*="查看"]` - 带"查看"标签的元素
- `[aria-label*="显示"]` - 带"显示"标签的元素
- `[title*="查看"]` - 带"查看"标题的元素
- `[title*="显示"]` - 带"显示"标题的元素

### 3. 容错处理
- 如果找不到眼睛图标，继续采集（可能已经显示）
- 如果点击失败，记录错误但不中断采集流程
- 在控制台输出详细的调试信息

## 使用方法

### 1. 重新加载插件
1. 打开 `chrome://extensions/`
2. 找到 "Zilo 达人采集助手"
3. 点击刷新按钮 🔄

### 2. 测试采集
1. 打开抖音达人详情页
2. **不需要手动点击眼睛图标**
3. 直接点击 "添加到 Zilo" 按钮
4. 插件会自动：
   - 点击眼睛图标
   - 等待联系方式显示
   - 提取并保存数据

### 3. 查看日志
按 F12 打开控制台，可以看到：
```
[Zilo] 正在查找眼睛图标...
[Zilo] 找到眼睛图标: [class*="eye"]
[Zilo] 已点击眼睛图标，等待联系方式显示...
[Zilo] 正在从页面提取联系方式...
[Zilo] 找到手机号: 15784254551(+8081)
[Zilo] 找到微信号: zzcdb2121
```

## 优势

1. **用户体验更好**：不需要手动点击眼睛图标
2. **操作更简单**：一键完成所有操作
3. **成功率更高**：自动等待确保数据加载完成
4. **容错性强**：即使找不到眼睛图标也能继续采集

## 注意事项

1. **等待时间**：设置为 2 秒，如果网络慢可能需要调整
2. **页面结构变化**：如果抖音修改页面结构，可能需要更新选择器
3. **调试信息**：所有操作都会在控制台输出，方便排查问题
