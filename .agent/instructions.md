# 工作区专项规则 (NEW BD)

## 技术规范
- **金额存储**：所有金额字段（如 `unitCost`, `salesGmv`, `pitFee` 等）必须在数据库中以整数存储（单位：分）。
- **前端风格**：采用 "Deep Space / Premium Glass" 现代视觉风格，强调玻璃拟态、渐变和阴影。
- **状态管理**：使用 Zustand 进行轻量级状态管理。
- **架构原则**：达人端口（Influencer Portal）必须保持完全的数据隔离。

## 业务逻辑
- **ROI 计算**：ROI = 销售 GMV / (样品成本 + 快递费 + 坑位费 + 实付佣金)。
- **回本判断**：ROI < 1 为亏损，ROI >= 1 为回本。
