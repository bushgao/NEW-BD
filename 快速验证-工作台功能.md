## 🔍 API 快速测试

打开浏览器控制台 (F12)，执行:

```javascript
// 获取 Token
const token = localStorage.getItem('token');

// 测试今日清单 API
fetch('http://localhost:3000/api/reports/my-dashboard/today-todos', {
  headers: { 'Authorization': `Bearer ${token}` }
})
.then(r => r.json())
.then(d => console.log('✅ 今日清单:', d.success ? '成功' : '失败'));

// 测试工作统计 API
fetch('http://localhost:3000/api/reports/my-dashboard/work-stats?period=today', {
  headers: { 'Authorization': `Bearer ${token}` }
})
.then(r => r.json())
.then(d => console.log('✅ 工作统计:', d.success ? '成功' : '失败'));

// 测试达人列表 API（快捷入口依赖）
fetch('http://localhost:3000/api/influencers?page=1&pageSize=1', {
  headers: { 'Authorization': `Bearer ${token}` }
})
.then(r => r.json())
.then(d => console.log('✅ 快捷入口:', d.success ? '成功' : '失败'));
```

**期望结果**: 所有 API 返回 `success: true`

## 🐛 快速问题排查

### 问题: API 返回 401

**解决**: 重新登录
```javascript
localStorage.removeItem('token');
// 然后在页面上重新登录
```

### 问题: 数据为空

**解决**: 创建测试数据
1. 添加几个达人
2. 创建几个合作
3. 设置截止日期
4. 刷新页面

### 问题: 组件未显示

**解决**: 检查页面和账号
1. 确保在 Dashboard 页面
2. 确保使用商务人员账号
3. 刷新页面 (Ctrl+F5)
