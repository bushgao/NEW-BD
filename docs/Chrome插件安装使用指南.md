# Zilo 达人采集助手 - 安装使用指南

## 📦 插件简介

Zilo 达人采集助手是一个 Chrome 浏览器插件，可以在抖音精选联盟页面一键采集达人信息到 Zilo 系统，大幅提升工作效率。

### 核心功能

- ✨ 自动识别达人详情页并注入采集按钮
- 🚀 一键提取达人昵称、抖音号、粉丝数、类目、等级等信息
- 📊 实时统计采集成功和失败数量
- 🔐 安全的 Token 认证机制
- 💾 配置信息本地存储

---

## 🚀 快速开始

### 第一步：安装插件

1. **打开 Chrome 扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或者：菜单 > 更多工具 > 扩展程序

2. **启用开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 点击开启

3. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 在弹出的文件选择窗口中，找到项目根目录下的 `chrome-extension` 文件夹
   - 选择该文件夹并确认

4. **确认安装成功**
   - 在扩展列表中看到"Zilo 达人采集助手"
   - 浏览器工具栏出现插件图标（紫色 Z 字母）

### 第二步：配置插件

#### 2.1 获取登录令牌

1. 打开 Zilo 系统：http://localhost:5173
2. 使用您的账号登录（工厂老板或商务人员账号）
3. 按键盘 `F12` 打开浏览器开发者工具
4. 切换到 `Application` 标签页
5. 在左侧菜单展开 `Local Storage`
6. 点击 `http://localhost:5173`
7. 在右侧列表中找到 `token` 字段
8. 双击 `token` 的值，复制整个字符串（通常很长）

#### 2.2 配置插件

1. 点击浏览器工具栏中的 Zilo 插件图标
2. 在弹出窗口中点击"⚙️ 配置设置"按钮
3. 填写配置信息：
   - **API 地址**：`http://localhost:3000/api`（默认值，通常不需要修改）
   - **登录令牌**：粘贴刚才复制的 token 值
4. 点击"💾 保存配置"按钮
5. 看到"配置已保存"提示后，点击"← 返回"

#### 2.3 验证配置

1. 在主界面查看"连接状态"
2. 如果显示"✓ 已连接"，说明配置成功
3. 如果显示"✗ 未配置"，请检查 token 是否正确

## 📖 使用方法

### 采集单个达人

1. **打开抖音精选联盟**
   - 访问：https://buyin.jinritemai.com
   - 使用您的抖音账号登录

2. **进入达人详情页**
   - 在达人广场搜索达人
   - 点击达人卡片进入详情页
   - 确保 URL 包含 `daren-profile`

3. **一键采集**
   - 页面右上角会自动出现"添加到 Zilo"按钮（紫色渐变）
   - 点击按钮
   - 等待 1-2 秒
   - 看到"✓ 已添加"提示表示成功

4. **验证结果**
   - 返回 Zilo 系统的"达人管理"页面
   - 刷新页面
   - 在列表中找到刚才采集的达人

### 查看统计信息

1. 点击浏览器工具栏中的 Zilo 插件图标
2. 在弹出窗口中查看：
   - **已采集**：成功采集的达人数量
   - **失败**：采集失败的次数
   - **连接状态**：与 Zilo 系统的连接状态

## ❓ 常见问题

### 1. 按钮没有出现？

**可能原因：**
- 不在达人详情页
- 页面还在加载中
- 插件未启用

**解决方法：**
- 确保 URL 包含 `buyin.jinritemai.com/dashboard/servicehall/daren-profile`
- 等待页面完全加载（3-5 秒）
- 刷新页面重试
- 检查插件是否启用：`chrome://extensions/`

### 2. 点击按钮后提示"采集失败"？

**可能原因：**
- Zilo 后端服务未运行
- Token 已过期
- 网络连接问题
- 达到达人数量上限

**解决方法：**
- 确认后端服务正在运行：访问 http://localhost:3000/health
- 重新获取 Token 并配置
- 检查网络连接
- 联系管理员升级套餐

### 3. 提示"请先在插件设置中配置登录令牌"？

**解决方法：**
- 按照"第二步：配置插件"的步骤重新配置
- 确保 Token 完整复制，没有多余的空格

### 4. 提示"已达到达人数量上限"？

**说明：**
- 您的套餐已达到达人数量限制
- 需要升级套餐才能继续添加

**解决方法：**
- 联系平台管理员
- 升级到更高级别的套餐

### 5. 采集的信息不完整？

**可能原因：**
- 抖音精选联盟页面结构变化
- 某些信息未公开显示

**解决方法：**
- 在 Zilo 系统中手动补充信息
- 联系开发团队更新插件

### 6. Token 在哪里找？

**详细步骤：**
1. 打开 Zilo 系统并登录
2. 按 `F12` 打开开发者工具
3. 点击 `Application` 标签
4. 左侧菜单：`Local Storage` > `http://localhost:5173`
5. 右侧列表找到 `token` 字段
6. 复制 `token` 的值（一长串字符串）

## 🔧 开发调试

### 查看插件日志

**后台日志（Background Script）：**
1. 打开 `chrome://extensions/`
2. 找到"Zilo 达人采集助手"
3. 点击"详细信息"
4. 点击"检查视图：Service Worker"
5. 在控制台查看日志

**内容脚本日志（Content Script）：**
1. 在达人详情页按 `F12`
2. 切换到 `Console` 标签
3. 查找以 `[Zilo]` 开头的日志

### 重新加载插件

修改代码后需要重新加载：
1. 打开 `chrome://extensions/`
2. 找到"Zilo 达人采集助手"
3. 点击刷新图标 🔄

### 测试采集功能

1. 打开测试达人页面
2. 打开浏览器控制台（F12）
3. 点击"添加到 Zilo"按钮
4. 观察控制台输出的日志
5. 检查网络请求（Network 标签）

## 💡 使用技巧

### 技巧 1：批量采集

虽然插件目前是单个采集，但可以通过以下方式提高效率：
1. 在抖音精选联盟打开多个达人详情页（新标签页）
2. 依次切换标签页，点击"添加到 Zilo"按钮
3. 所有采集完成后，在 Zilo 系统中统一查看

### 技巧 2：快速验证

采集后想快速验证是否成功：
1. 点击插件图标
2. 查看"已采集"数量是否增加
3. 如果增加，说明采集成功

### 技巧 3：定期更新 Token

Token 可能会过期，建议：
- 每周更新一次 Token
- 如果采集失败，首先尝试更新 Token
