## 🔧 数据库管理

### Prisma Studio（可视化数据库工具）

```bash
cd packages/backend
npm run db:studio
```

这将打开 Prisma Studio，可以直接查看和编辑数据库数据。

### 重置数据库

如果需要重置数据库到初始状态：

```bash
cd packages/backend
npm run db:migrate
npm run db:seed
```

## 📝 注意事项

1. **密码安全**: 这些是测试账号，生产环境请使用强密码
2. **数据隔离**: 不同工厂的数据是完全隔离的
3. **权限控制**: 商务人员无法访问工厂老板的功能

## 🆘 遇到问题？

### 无法登录
- 检查邮箱和密码是否正确
- 确认后端服务是否正常运行
- 查看浏览器控制台的错误信息

### 数据不显示
- 确认数据库是否已运行 seed
- 检查后端日志是否有错误
- 使用 Prisma Studio 查看数据库数据

### 服务无法启动
- 检查端口是否被占用（3000, 5173）
- 确认 PostgreSQL 是否正在运行
- 查看 .env 文件配置是否正确
