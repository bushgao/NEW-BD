# 快速跟进功能 - 快速使用指南

## 🎯 功能简介

快速跟进功能让商务人员能够更高效地记录跟进信息，支持：
- ⚡ 模板快速填充
- 🎤 语音输入（Chrome浏览器）
- 📷 图片上传
- ⏰ 自动记录时间

---

## 📍 功能位置

**合作管道页面** → **合作卡片** → **⚡ 快速跟进按钮**

在每个合作卡片的底部右侧，可以看到一个蓝色的闪电图标按钮，这就是快速跟进入口。

---

## 🚀 快速开始

### 方式一：使用模板（最快）

1. 点击合作卡片上的 **⚡ 快速跟进** 按钮
2. 在弹窗顶部找到合适的模板标签
3. 点击模板标签，内容自动填充
4. 可以继续编辑或直接点击 **发送跟进**

**示例**：
```
场景：刚寄出样品，需要通知达人
操作：点击"样品确认"模板 → 修改天数 → 发送
```

### 方式二：语音输入（最方便）

1. 点击 **⚡ 快速跟进** 按钮
2. 点击 **语音输入** 按钮
3. 对着麦克风说话
4. 识别结果自动填入
5. 点击 **发送跟进**

**注意**：
- 需要使用 Chrome 或 Edge 浏览器
- 首次使用需要授权麦克风权限
- 支持中文识别

### 方式三：上传图片（最直观）

1. 点击 **⚡ 快速跟进** 按钮
2. 输入文字说明
3. 点击 **上传图片** 按钮
4. 选择图片文件
5. 预览确认后点击 **发送跟进**

**适用场景**：
- 发送报价单截图
- 展示样品照片
- 分享聊天记录

## 💡 使用技巧

```
1. 点击"报价跟进"模板
```

点击多个模板，内容会自动追加，不会覆盖：
```
点击"礼貌催促" → 内容填充
点击"感谢回复" → 内容追加

```

### 技巧3：快捷键
- **Tab**: 在按钮间切换
- **Enter**: 提交表单
- **Esc**: 关闭弹窗

## 🎬 完整操作流程

### 场景：样品已寄出，通知达人

1. **打开快速跟进**
   - 在合作管道找到对应的合作
   - 点击卡片上的 ⚡ 按钮

2. **选择模板**
   - 找到"寄样阶段"分类
   - 点击"样品确认"模板

3. **编辑内容**
   - 自动填充：`您好，样品已经寄出，预计{天数}天内送达...`
   - 修改 `{天数}` 为实际天数，如 `3`

4. **添加图片**（可选）
   - 点击"上传图片"
   - 选择快递单照片
   - 预览确认

5. **发送**
   - 点击"发送跟进"
   - 等待成功提示
   - 弹窗自动关闭

6. **验证**
   - 合作卡片的跟进数量 +1
   - 点击跟进记录按钮可查看详情

## ⚠️ 注意事项

### 语音输入
- ✅ Chrome/Edge 浏览器支持
- ❌ Firefox 不支持
- ⚠️ Safari 部分支持
- 需要授权麦克风权限
- 需要安静的环境

### 图片上传
- 支持 jpg、png、gif 等常见格式
- 建议单张图片不超过 5MB
- 可以上传多张图片
- 点击图片右上角 ✕ 可删除

### 内容限制
- 最多 1000 字
- 不能为空
- 支持换行和特殊字符

## 🔧 常见问题

### Q1: 语音输入没反应？
**A**: 
1. 检查是否使用 Chrome/Edge 浏览器
2. 检查麦克风权限是否授权
3. 检查麦克风是否正常工作
4. 尝试刷新页面重试

### Q2: 模板内容如何修改？
**A**: 
- 点击模板后，内容会填充到输入框
- 直接在输入框中编辑即可
- 可以删除、修改、追加内容

### Q3: 可以保存自定义模板吗？
**A**: 
- 当前版本暂不支持
- 后续版本会添加此功能
- 可以先在备忘录保存常用话术

### Q4: 图片上传失败？
**A**: 
- 检查图片格式是否支持
- 检查图片大小是否过大
- 检查网络连接是否正常
- 尝试压缩图片后重试

### Q5: 跟进记录在哪里查看？
**A**: 
- 点击合作卡片上的 💬 按钮
- 或点击卡片进入详情页
- 在跟进记录标签页查看
