# 🚀 快速验证 - 数据录入功能

**验证时间**: 5-10分钟  
**目标**: 快速验证所有数据录入优化功能是否正常工作

---

## 📱 方法一: 使用测试页面（最快）

### 1. 打开测试页面
在浏览器中打开: `test-checkpoint-task31.html`

### 2. 加载 Token
点击 **"从 LocalStorage 加载"** 按钮

### 3. 运行测试
点击 **"运行所有测试"** 按钮

### 4. 查看结果
- ✅ 绿色 = 通过
- ❌ 红色 = 失败
- 查看汇总统计

**预期结果**: 所有测试通过（或大部分通过）

---

## 🖱️ 方法二: 手动快速验证（5分钟）

### 1️⃣ 智能表单（1分钟）

1. 打开 http://localhost:5173/app/pipeline
2. 点击 **"创建合作"** 按钮
3. 选择一个达人
4. **观察**: 表单是否显示智能推荐？

✅ **通过标准**: 看到样品推荐、价格推荐

---

### 2️⃣ 批量操作（1分钟）

1. 在合作管道页面
2. 勾选 2-3 个合作
3. 点击 **"批量操作"** 按钮
4. 选择 **"批量更新状态"**
5. 选择目标状态并确认

✅ **通过标准**: 操作成功，显示成功/失败统计

---

### 3️⃣ 数据验证（1分钟）

1. 在创建合作表单中
2. 填写一个 **负数价格**（如 -100）
3. **观察**: 是否显示错误提示？
4. 尝试提交

✅ **通过标准**: 显示错误提示，阻止提交

---

### 4️⃣ 表单缓存（1分钟）

1. 在创建合作表单中填写部分数据
2. **不提交**，直接刷新页面（F5）
3. **观察**: 数据是否恢复？

✅ **通过标准**: 刷新后数据自动恢复

---

### 5️⃣ 智能推荐（1分钟）

1. 在创建合作表单中
2. 选择一个有历史合作的达人
3. 点击样品输入框
4. **观察**: 是否显示推荐样品？

✅ **通过标准**: 显示历史ROI最高的样品推荐

## 🐛 如果发现问题

### 问题 1: 智能推荐不显示
**可能原因**:
- 达人没有历史合作数据
- API 未正确调用

**解决方法**:
1. 打开浏览器控制台（F12）
2. 查看 Network 标签
3. 检查 `/api/collaborations/suggestions` 请求
4. 查看响应数据

### 问题 2: 批量操作失败
**可能原因**:
- 权限不足
- 数据状态不符合要求

**解决方法**:
1. 检查用户权限
2. 确认选中的合作状态
3. 查看错误提示信息

### 问题 3: 表单缓存不工作
**可能原因**:
- 浏览器不支持 localforage
- 缓存被清理

**解决方法**:
1. 检查浏览器控制台是否有错误
2. 确认 localforage 库已加载
3. 检查浏览器存储设置
