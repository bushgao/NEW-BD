## 🎯 功能说明

### 快捷操作卡片

快捷操作面板包含三个主要卡片：

#### 1. 超期合作 (红色)
- **图标**: 🕐 时钟图标
- **说明**: 显示已超过截止日期但尚未完成的合作数量
- **统计规则**: 
  - deadline < 当前时间
  - stage != PUBLISHED 且 != REVIEWED
- **点击操作**: 跳转到合作管道页面，自动筛选超期合作

#### 2. 待签收样品 (橙色)
- **图标**: 📦 包裹图标
- **说明**: 显示寄出超过7天但达人尚未签收的样品数量
- **统计规则**:
  - receivedStatus = PENDING
  - dispatchedAt < (当前时间 - 7天)
- **点击操作**: 跳转到样品管理页面，自动筛选待签收样品

#### 3. 待录入结果 (蓝色)
- **图标**: 📝 文档图标
- **说明**: 显示已上车超过14天但尚未录入结果的合作数量
- **统计规则**:
  - stage = SCHEDULED 或 PUBLISHED
  - result = null
  - 存在 onboardStatus = ONBOARD 且 dispatchedAt < (当前时间 - 14天) 的样品
- **点击操作**: 跳转到结果录入页面，自动筛选待录入结果

### 导出报表按钮

- **位置**: 面板右上角
- **图标**: 📥 下载图标
- **功能**: 导出当前 Dashboard 的数据报表
- **状态**: 开发中（目前显示提示信息）

## 💡 使用场景

### 场景 1: 每日工作检查

**步骤**:
1. 登录系统，查看 Dashboard
2. 查看快捷操作面板的三个数字
3. 优先处理红色的超期合作
4. 然后处理橙色的待签收样品
5. 最后处理蓝色的待录入结果

**建议**: 每天早上第一件事就是查看快捷操作面板，了解当天需要处理的重要事项。

### 场景 2: 快速定位问题

**步骤**:
1. 发现某个卡片数字较大（如超期合作有10个）
2. 点击该卡片
3. 系统自动跳转到对应页面并应用筛选
4. 查看详细列表，逐个处理

**优势**: 无需手动筛选，一键直达问题列表。

### 场景 3: 团队管理

**步骤**:
1. 查看快捷操作面板的数据
2. 如果某项数字持续偏高，说明团队在该环节存在问题
3. 可以结合商务团队工作进展表格，定位具体负责人
4. 进行针对性的指导和改进

**价值**: 帮助工厂老板快速发现团队工作中的瓶颈。

## 🎨 视觉设计

### 颜色含义

| 颜色 | 含义 | 优先级 |
|------|------|--------|
| 🔴 红色 | 超期合作 | 高 - 需要立即处理 |
| 🟠 橙色 | 待签收样品 | 中 - 需要及时跟进 |
| 🔵 蓝色 | 待录入结果 | 正常 - 按计划处理 |

### 数量显示

- **0**: 灰色 Badge，表示无待办事项
- **1-5**: 对应颜色 Badge，正常范围
- **6-10**: 对应颜色 Badge，需要关注
- **>10**: 对应颜色 Badge，需要重点处理

### 卡片状态

- **正常**: 白色背景，灰色边框
- **悬停**: 边框高亮为对应颜色
- **点击**: 跳转到对应页面

## 🔧 常见问题

### Q1: 为什么快捷操作面板不显示？

**A**: 可能的原因：
1. 您不是工厂老板或平台管理员
2. 数据加载失败，请刷新页面重试
3. 浏览器缓存问题，请清除缓存后重试

### Q2: 数字显示为0是正常的吗？

**A**: 是的，如果所有待办事项都已处理完成，数字会显示为0。这是一个好现象，说明团队工作进展顺利。

### Q3: 点击卡片后没有自动筛选怎么办？

**A**: 这个功能目前正在开发中，暂时需要手动在目标页面进行筛选。后续版本会完善自动筛选功能。

### Q4: 导出报表按钮点击后没有反应？

**A**: 导出报表功能目前正在开发中，点击后会显示提示信息。后续版本会实现完整的导出功能。

### Q5: 如何修改统计规则？

**A**: 统计规则是系统预设的，基于业务最佳实践。如需修改，请联系技术团队。

## 📈 数据解读

### 超期合作数量

- **0-2**: 优秀，团队按时完成工作
- **3-5**: 良好，有少量延期但在可控范围
- **6-10**: 需要关注，可能存在资源不足或流程问题
- **>10**: 需要重点处理，建议召开团队会议分析原因

### 待签收样品数量

- **0-3**: 正常，达人签收及时
- **4-6**: 需要关注，可能需要提醒达人
- **7-10**: 需要跟进，建议主动联系达人
- **>10**: 需要重点处理，可能存在物流或沟通问题

### 待录入结果数量

- **0-2**: 优秀，结果录入及时
- **3-5**: 良好，有少量延迟但在可控范围
- **6-10**: 需要关注，建议提醒商务及时录入
- **>10**: 需要重点处理，可能影响数据分析准确性
