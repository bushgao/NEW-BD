# 重启后端服务

## 问题

从浏览器截图看到仍然有400错误，可能是：
1. 后端服务没有重启，enrichUserData中间件没有生效
2. Token仍然是旧的

## 解决步骤

### 第1步：重启后端服务

```bash
# 进入后端目录
cd packages/backend

# 停止当前服务（如果在运行）
# 按 Ctrl+C 停止

# 重新启动
npm run dev
```

### 第2步：清除浏览器Token

在浏览器控制台（F12）执行：

```javascript
localStorage.clear();
sessionStorage.clear();
location.reload();
```

### 第3步：重新登录

1. 访问：http://localhost:5173
2. 登录：pinpai001@gmail.com
3. 查看Dashboard

## 验证

重新登录后，检查浏览器控制台：
- ✅ 应该没有400错误
- ✅ API请求应该返回200
- ✅ Dashboard应该显示数据
