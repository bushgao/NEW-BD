# 莫兰迪配色 + 尺寸优化完成

## 🎨 莫兰迪配色方案

### 什么是莫兰迪色系?
莫兰迪色系源自意大利画家乔治·莫兰迪的作品,特点是:
- **低饱和度** - 柔和不刺眼
- **高级灰调** - 优雅专业
- **视觉舒适** - 长时间使用不疲劳

### 新配色方案

#### 阶段颜色 (Pipeline Stages)
```
LEAD (线索)        → #B8B8B8  浅灰 - 柔和中性
CONTACTED (已建联) → #8EACBB  雾霾蓝 - 柔和蓝
QUOTED (已报价)    → #A89BB9  薰衣草紫 - 柔和紫
SAMPLED (已寄样)   → #D4A574  驼色 - 柔和橙
SCHEDULED (已排期) → #7FA99B  鼠尾草绿 - 柔和青
PUBLISHED (已发布) → #9CAF88  橄榄绿 - 柔和绿
REVIEWED (已复盘)  → #C89B9C  豆沙粉 - 柔和粉
```

#### 功能颜色
```
成功/GMV    → #9CAF88  橄榄绿
警告/超期   → #C89B9C  豆沙粉
提示/待办   → #D4A574  驼色
信息/建联   → #8EACBB  雾霾蓝
紫色/ROI    → #A89BB9  薰衣草紫
中性/默认   → #B8B8B8  浅灰
```

## 📏 尺寸优化

### Progress圆圈 (管道分布)
- **旧版**: 70-80px 直径
- **新版**: 60px 直径
- **线条**: 8px 粗细
- **间距**: 减小到6px
- **字体**: 11px

**效果**: 更紧凑,一屏显示更多内容

### 卡片优化
- 统一圆角: 12px
- 统一阴影: `0 2px 8px rgba(0,0,0,0.08)`
- 统一内边距

## 💡 设计理念

### 为什么选择莫兰迪色系?

1. **专业性** - 企业级应用需要稳重
2. **舒适性** - 长时间使用不疲劳
3. **高级感** - 低饱和度更显品质
4. **和谐性** - 颜色之间不冲突
5. **现代感** - 符合当代审美

### 配色原则

- **主色调**: 灰蓝、灰绿系
- **强调色**: 豆沙粉、驼色
- **中性色**: 浅灰
- **饱和度**: 30-50%
- **明度**: 60-80%
