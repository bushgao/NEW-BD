# 在服务器上初始化测试数据

## 步骤：

1. 登录服务器终端（宝塔面板 → 终端）

2. 运行以下命令：

cd /www/wwwroot/zilo/packages/backend
npm run db:seed

3. 等待完成，看到 "🎉 Seed completed successfully!" 表示成功
