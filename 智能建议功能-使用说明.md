# 智能建议功能 - 使用说明

## 🎯 功能简介

智能建议功能会在您创建合作时，根据历史数据自动推荐：
- 📦 **样品** - 推荐效果最好的样品
- 💰 **报价** - 推荐合理的报价金额
- 📅 **排期** - 推荐最佳发布时间

## 🚀 如何使用

### 1. 打开新建合作对话框

在"合作管道"页面，点击"新建合作"按钮

### 2. 选择达人

在"选择达人"下拉框中选择一个达人

### 3. 查看智能建议

选择达人后，会自动显示"智能建议"卡片，包含：

```
💡 智能建议
┌─────────────────────────────────────┐
│ [强烈推荐] 测试样品A（历史最佳）      │
│ 💡 该达人使用此样品平均GMV为 ¥5000  │
│                    [应用] [忽略]     │
├─────────────────────────────────────┤
│ [强烈推荐] ¥1500（历史平均）         │
│ 💡 该达人历史平均报价为 ¥1500       │
│                    [应用] [忽略]     │
├─────────────────────────────────────┤
│ [建议] 一周后 20:00（黄金时段）      │
│ 💡 黄金时段，用户活跃度最高          │
│                    [应用] [忽略]     │
└─────────────────────────────────────┘
```

### 4. 应用建议

- 点击"应用"按钮，建议会自动填充到对应字段
- 点击"忽略"按钮，移除该建议

### 5. 完成创建

填写其他必填信息，点击"创建"按钮

## 🏷️ 置信度说明

建议会标注不同的置信度：

- **强烈推荐** (绿色) - 基于该达人的历史数据，准确度高
- **建议** (橙色) - 基于同类达人的数据，准确度中等
- **可选** (灰色) - 基于通用规则，仅供参考

## 💾 草稿自动保存

表单内容会自动保存为草稿：

- 填写内容后 1 秒自动保存
- 关闭对话框时保存
- 重新打开时自动恢复
- 草稿保存 24 小时

## ❓ 常见问题

### Q: 为什么有些达人没有建议？

**A**: 这是正常的！可能的原因：
- 新添加的达人，没有历史合作记录
- 该达人从未成功合作过
- 数据库中没有足够的数据

### Q: 建议准确吗？

**A**: 建议基于历史数据和算法计算：
- "强烈推荐"的建议准确度最高
- "建议"的建议准确度中等
- "可选"的建议仅供参考

您可以根据实际情况选择是否采纳。

### Q: 可以修改建议吗？

**A**: 可以！
- 应用建议后，可以手动修改字段值
- 也可以完全忽略建议，手动填写

### Q: 草稿会丢失吗？

**A**: 不会！
- 草稿自动保存在浏览器本地
- 保存时间为 24 小时
- 即使关闭浏览器也不会丢失

## 🧪 测试功能

如果您想测试智能建议功能是否正常工作：

1. 打开浏览器控制台（F12）
2. 选择一个达人
3. 查看控制台日志，应该看到：
   ```
   handleInfluencerChange: 达人选择变化
   loadSuggestions: 开始加载建议
   loadSuggestions: 合并后的建议数量: X
   ```

或者使用测试页面：
```
打开: test-suggestions-direct.html
```

如果遇到问题：

2. 检查网络连接
3. 查看浏览器控制台是否有错误
